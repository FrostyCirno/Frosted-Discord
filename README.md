# Frosted-Discord

A collection of Discord themes for use with Vencord.

---

> **Base theme:**  
> https://github.com/refact0r/midnight-discord

---

## How to Use
1. Open Vencord.
2. Go to the **Online Themes** section.
3. Paste any of the links below to apply a theme.
